# Микита Добрецов — навчальний сайт

Це статичний сайт з **трьох HTML-сторінок**:
- `index.html` — інформація про студента
- `resume.html` — резюме (українською)
- `report.html` — аналітичний звіт (США): 5 загроз + 5 рекомендацій

## Як завантажити на GitHub Pages (швидко)

1. Створіть репозиторій на GitHub (наприклад, `my-site`).
2. Завантажте **вміст цієї папки** у корінь репозиторію (файли `index.html`, `resume.html`, `report.html`, папка `assets/`).
3. Відкрийте репозиторій → **Settings** → **Pages**.
4. У блоці **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main` (або `master`) / Folder: `/root`
5. Збережіть. Посилання на сайт з’явиться в Pages.

> Якщо це репозиторій формату `username.github.io`, то сайт буде на `https://username.github.io/`.
